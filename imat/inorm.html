  function inorm.m

  Author


Jaroslav Horacek

  Description


computes the norm of an interval matrix

  Input parameters


iA ... interval matrix
type ... string characterizing norm type
1 - column norm (default)
2 - Euclid norm
'inf' - row norm
'col' - column norm

  Output parameters


mn ... matrix norm according to its type
ver ... 1 if the result is verified, 0 otherwise
